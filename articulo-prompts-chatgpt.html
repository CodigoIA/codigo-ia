Los 10 mejores prompts de IA para productividad

Prompt #1 — Resumir cualquier documento largo

Ideal para analizar contratos, informes, artículos académicos o cualquier texto extenso en segundos.

📄 Análisis de documentos
"Analiza el siguiente texto y dame: 1) Un resumen ejecutivo de 3 párrafos, 2) Los 5 puntos más importantes en bullets, 3) Cualquier dato numérico o estadística relevante, 4) Conclusión y recomendación de acción. Texto: [pega aquí tu documento]"
ChatGPT / Claude / Gemini

Prompt #2 — Crear contenido para redes sociales

Genera una semana completa de contenido para Instagram, LinkedIn o Twitter en menos de 2 minutos.

📱 Redes sociales
"Actúa como un experto en marketing de contenidos. Crea 7 publicaciones para [red social] sobre [tema/nicho]. Cada publicación debe tener: título llamativo, texto de 100-150 palabras, llamada a la acción, y 5 hashtags relevantes. Tono: [profesional/casual/inspiracional]. Audiencia: [describe tu audiencia]."
ChatGPT / Claude

Prompt #3 — Responder emails difíciles

Para situaciones complicadas: rechazar propuestas, negociar precios, manejar quejas o dar malas noticias.

📧 Emails profesionales
"Necesito responder este email de forma [profesional/diplomática/firme]. El contexto es: [explica la situación]. El email original dice: [pega el email]. La respuesta debe: lograr [tu objetivo], mantener la relación, y ser de máximo 200 palabras. Dame 3 versiones con diferente tono."
Claude / ChatGPT

Prompt #4 — Analizar datos y generar insights

Convierte datos crudos en insights accionables sin necesitar ser analista de datos.

📊 Análisis de datos
"Eres un analista de datos senior. Analiza estos datos: [pega tus datos]. Dame: 1) Las 3 tendencias principales, 2) Anomalías o datos atípicos, 3) Correlaciones importantes, 4) Predicción para el próximo período, 5) Top 3 recomendaciones accionables con métricas específicas."
ChatGPT / Gemini
Publicidad

Prompt #5 — Crear plan de negocio

💼 Negocios
"Crea un plan de negocio ejecutivo para [idea de negocio] en el mercado de [país/región]. Incluye: propuesta de valor única, modelo de monetización, análisis de competencia con 3 competidores, estrategia de adquisición de clientes, proyección financiera del año 1, y roadmap de 90 días con hitos específicos."
Claude / ChatGPT

Prompt #6 — Mejorar cualquier texto

✍️ Escritura
"Mejora este texto manteniendo mi voz y estilo. Objetivos: hacerlo más claro, más persuasivo y más fácil de leer. Elimina redundancias, mejora la estructura y usa palabras más precisas. Al final, explica los 3 cambios más importantes que hiciste y por qué. Texto original: [pega tu texto]"
Claude / ChatGPT

Prompt #7 — Preparar para entrevistas

🎯 Carrera
"Actúa como un entrevistador senior de [empresa/industria] para el puesto de [cargo]. Hazme una entrevista simulada con 10 preguntas difíciles y realistas. Después de cada respuesta mía, dame feedback detallado sobre qué estuvo bien, qué mejorar y cómo responderías tú de forma ideal."
Claude / ChatGPT

Prompt #8 — Generar ideas creativas

💡 Creatividad
"Usa técnicas de brainstorming para generar 20 ideas sobre [tema/problema]. Para cada idea: nombre, descripción en una línea, nivel de viabilidad (1-10), y por qué sería única. Después selecciona las 3 mejores y desarrolla cada una en un párrafo con próximos pasos concretos."
ChatGPT / Claude / Gemini

Prompt #9 — Depurar y mejorar código

💻 Programación
"Eres un senior developer experto en [lenguaje]. Revisa este código: [pega tu código]. Identifica: 1) Bugs o errores, 2) Problemas de rendimiento, 3) Vulnerabilidades de seguridad, 4) Código que no sigue buenas prácticas. Luego dame la versión mejorada con comentarios explicando cada cambio."
Claude / ChatGPT / Gemini

Prompt #10 — Crear estrategia de contenido SEO

🔍 SEO
"Crea una estrategia de contenido SEO para un sitio web sobre [nicho/tema] dirigido a [audiencia] en [país]. Incluye: 20 palabras clave principales con intención de búsqueda, 10 títulos de artículos con alto potencial de tráfico, estructura de pillar pages y cluster pages, y un calendario editorial para los próximos 3 meses."
ChatGPT / Claude
🚀 Tip final: El secreto de un buen prompt es dar contexto, especificar el formato de salida y pedir múltiples opciones. Cuanto más específico seas, mejores resultados obtendrás de cualquier IA.
